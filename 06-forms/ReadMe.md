# 06 – Forms in HTML

This topic explains how to create and use HTML forms to collect user input
on a web page. Forms are essential for interactive websites.

## File Included

- `06-forms.html`  
  Demonstrates various form elements used to collect user data.

## What Are HTML Forms?

HTML forms are used to gather information from users. They contain
form controls such as text fields, radio buttons, checkboxes, drop-down
lists, and more. The data entered can be sent to a server for
processing using form attributes like `action` and `method` (GET or POST). :contentReference[oaicite:1]{index=1}

## Form Elements Covered

- `<form>` – Container for form elements.
- `<input type="text">` – Single-line text box.
- `<input type="email">` – Email input.
- `<input type="password">` – Password input.
- Radio buttons – Single choice options.
- Checkbox – Multiple choice option.
- `<select>` – Drop-down list.
- `<option>` – Item inside a drop-down list.
- `<textarea>` – Multi-line text box.
- Submit button – Sends the form data. :contentReference[oaicite:2]{index=2}

## How to Use

1. Open `06-forms.html` in a web browser.
2. Try filling out each field.
3. Click “Submit” to simulate data submission.

## Form Attributes

- `action` – URL where form data is sent (can be a server script).
- `method` – HTTP method for data submission (`GET` or `POST`). :contentReference[oaicite:3]{index=3}

- How form data can be collected from users.
