# 04 – Lists in HTML

This topic explains the different types of lists available in HTML.
Lists are used to organize related information in a structured way.

## File Included

- `04-lists.html`  
  Demonstrates all major types of HTML lists.

## Types of Lists

### Unordered List
- Created using the `<ul>` tag
- List items are marked with bullets

### Ordered List
- Created using the `<ol>` tag
- List items are numbered or ordered

### Description List
- Created using `<dl>`
- `<dt>` defines the term
- `<dd>` describes the term

### Nested List
- A list inside another list
- Used to represent sub-items or categories

## Common Tags Used

- `<ul>` – Unordered list
- `<ol>` – Ordered list
- `<li>` – List item
- `<dl>` – Description list
- `<dt>` – Description term
- `<dd>` – Description details

## How to Use

1. Open `04-lists.html` in a web browser.
2. Observe different list styles.
3. Modify list items to practice.
