## ðŸ“˜ Overview
Meta elements provide **information about the HTML document**.  
They are placed inside the `<head>` section and are not visible on the web page.

Meta tags help browsers, search engines, and devices understand the page.

## ðŸ§© Common Meta Elements

### `<meta charset>`
Defines the character encoding of the document.
Example: UTF-8 supports most languages.

### `<meta name="viewport">`
Controls how the page is displayed on different screen sizes (mobile-friendly).

### `<meta name="description">`
Provides a short description of the webpage for search engines.

### `<meta name="keywords">`
Specifies keywords related to the page content (less important today).

### `<meta name="author">`
Defines the author of the document.
